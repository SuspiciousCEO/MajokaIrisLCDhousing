# 精度はそれなりだよ。

定規と目分量でサイズ寸法測っているのでだいたいのなりゆきだよ。

外したネジをそのまま流用できるようにネジ穴の位置は意識しました。

ネジが入りにくい場合は、ハンドドリルなどで穴を少し拡張してみてね。

なにかに引っかけやすいように結束バンド用の穴も追加してあります。
